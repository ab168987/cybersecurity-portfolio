# Network Scanning & Enumeration

## Objective
The goal of this project was to understand how systems expose services on a network and how attackers identify potential entry points.

## Tools Used
- Nmap
- TryHackMe lab environment

## Process
- Performed a basic network scan to identify open ports
- Observed which services were running on those ports
- Reviewed how attackers gather system information

## Key Findings
- Open ports can reveal critical services such as SSH and HTTP
- Unnecessary open services increase security risks
- Even simple scans can expose valuable system information

## What I Learned
- How network scanning works
- Why limiting exposed services is important
- How attackers perform reconnaissance
